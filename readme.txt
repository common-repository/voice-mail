== Description ==
Voice Mail is a wordpress plugin that allow users contact webmaster via voice mail.

Once installed the voice mail plugin, just insert the shortcode 'voice_mail' in your post or pages. 

In front end, the plugin will show a recoder and users can click 'record' button to start record voice mail, then click 'stop' and 'submit' to send voice message to webmaster.

Then the voice mail plugin will save user's voice message in plugin 'files' folder, and send a mail to webmaster, with the link of the voice message, webmaster can click the link check voice message. 
   
== Installation ==

1:Upload the Voice Mail plugin to your wordpress site
2:Activate the Voice Mail plugin
3:insert the shortcode 'voice_mail' in your post or pages.

== How To Use ==
Once installed the voice mail plugin, just insert the shortcode 'voice_mail' in your post or pages.
If the user send voice message to you,you will get a mail with the link of the voice message, webmaster can click the link check voice message.
